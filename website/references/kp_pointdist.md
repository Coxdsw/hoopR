# `kp_pointdist`

Get Team Points Distribution


## Description

Get Team Points Distribution


## Usage

```r
kp_pointdist(min_year, max_year)
```


## Arguments

Argument      |Description
------------- |----------------
`min_year`     |     First year of data to pull
`max_year`     |     Last year of data to pull


## Examples

```r
kp_pointdist(min_year = 2020, max_year = 2020)
```
