# `kp_fourfactors`

Get Four Factors Data


## Description

Get Four Factors Data


## Usage

```r
kp_fourfactors(min_year, max_year)
```


## Arguments

Argument      |Description
------------- |----------------
`min_year`     |     First year of data to pull
`max_year`     |     Last year of data to pull


## Examples

```r
kp_fourfactors(min_year = 2020, max_year = 2020)
```
