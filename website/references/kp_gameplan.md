# `kp_gameplan`

Get KenPom's team game plan page


## Description

Get KenPom's team game plan page


## Usage

```r
kp_gameplan(team, year = 2020)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.
`year`     |     Year of data to pull


## Examples

```r
kp_gameplan(team='Florida St.', year=2020)
```
