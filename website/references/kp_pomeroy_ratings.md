# `kp_pomeroy_ratings`

Get KenPom Ratings


## Description

Get KenPom Ratings


## Usage

```r
kp_pomeroy_ratings(min_year, max_year)
```


## Arguments

Argument      |Description
------------- |----------------
`min_year`     |     First year of data to pull
`max_year`     |     Last year of data to pull


## Examples

```r
kp_pomeroy_ratings(min_year = 2020, max_year = 2020)
```
