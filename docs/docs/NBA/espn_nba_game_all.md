## Description

Get ESPN NBA game data (Pbp, Team and Player Box)


## Usage

```r
espn_nba_game_all(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_nba_game_all(game_id = 401283399)
```
