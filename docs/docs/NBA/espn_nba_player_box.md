# `espn_nba_player_box`

## Description

Get ESPN NBA player box scores


## Usage

```r
espn_nba_player_box(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_nba_player_box(game_id = 401283399)
```
