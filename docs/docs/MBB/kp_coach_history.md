# `kp_coach_history`

## Description

Get KenPom's coaching resume data


## Usage

```r
kp_coach_history(coach)
```


## Arguments

Argument      |Description
------------- |----------------
`coach`     |     Coach filter to select.


## Value

A data frame with 30 columns

## Examples

```r
kp_coach_history(coach = 'Leonard Hamilton')
```
