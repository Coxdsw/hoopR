# `espn_mbb_game_all`

## Description

Get ESPN men's college basketball data (Pbp, Team and Player Box)


## Usage

```r
espn_mbb_game_all(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID


## Examples

```r
espn_mbb_game_all(game_id = 401256760)
```
