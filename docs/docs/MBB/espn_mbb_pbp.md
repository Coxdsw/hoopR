# `espn_mbb_pbp`

## Description

Get ESPN men's college basketball PBP data

## Usage

```r
espn_mbb_pbp(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID


## Examples

```r
espn_mbb_pbp(game_id = 401256760)
```
