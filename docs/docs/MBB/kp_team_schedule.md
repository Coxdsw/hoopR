# `kp_team_schedule`

## Description

Get team schedule results


## Usage

```r
kp_team_schedule(team, year)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.
`year`     |     Year of data to pull


## Examples

```r
kp_team_schedule(team = 'Florida St.', year= 2020)
```
