# `espn_mbb_standings`

## Description

Get ESPN men's college basketball standings

## Usage

```r
espn_mbb_standings(year)
```

## Arguments

Argument      |Description
------------- |----------------
`year`     |     Either numeric or character (YYYY)


## Examples

```r
espn_mbb_standings(2021)
```
