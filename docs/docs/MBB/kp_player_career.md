# `kp_player_career`

## Description

Get KenPom's player career stats from the player page


## Usage

```r
kp_player_career(player_id)
```


## Arguments

Argument      |Description
------------- |----------------
`player_id`     |     Player Id filter to select.


## Examples

```r
kp_player_career(player_id = '41180')
```
