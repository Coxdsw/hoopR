# `kp_kpoy`

## Description

Get KPoY Leaders Tables


## Usage

```r
kp_kpoy(year)
```


## Arguments

Argument      |Description
------------- |----------------
`year`     |     Year of data to pull (earliest year of data available: 2011)


## Examples

```r
kp_kpoy(year=2020)
```
