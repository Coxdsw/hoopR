# `espn_mbb_team_box`

## Description

Get ESPN men's college basketball team box scores


## Usage

```r
espn_mbb_team_box(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_mbb_team_box(game_id = 401256760)
```
