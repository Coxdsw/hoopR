# `kp_team_player_stats`

## Description

Get Team Player Stats

## Usage

```r
kp_team_player_stats(team, year = 2020)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.
`year`     |     Year of data to pull


## Examples

```r
kp_team_player_stats(team = 'Florida St.', year = 2020)
```
