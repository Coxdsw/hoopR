# `kp_team_history`

## Description

Get KenPom's team efficiency history


## Usage

```r
kp_team_history(team)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.


## Value

A data frame with 30 columns

## Examples

```r
kp_team_history(team = 'Florida St.')
```
