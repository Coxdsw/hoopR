# `espn_mbb_player_box`

## Description

Get ESPN men's college basketball player box scores

## Usage

```r
espn_mbb_player_box(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_mbb_player_box(game_id = 401256760)
```
