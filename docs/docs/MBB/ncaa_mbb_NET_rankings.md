# `ncaa_mbb_NET_rankings`

## Description

Get men's college basketball NET rankings for the current date from the NCAA website


## Usage

```r
ncaa_mbb_NET_rankings()
```


## Value

Returns a tibble

## Examples

```r
# Get current NCAA NET rankings
ncaa_mbb_NET_rankings()
```
