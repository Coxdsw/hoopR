# `kp_team_players`

## Description

Get KenPom's player stats from the team page


## Usage

```r
kp_team_players(team, year = 2020)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.
`year`     |     Year of data to pull


## Examples

```r
kp_team_players(team = 'Florida St.', year= 2020)
```
