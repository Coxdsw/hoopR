# `kp_minutes_matrix`

## Description

Get Minutes Matrix from Expanded Player Page


## Usage

```r
kp_minutes_matrix(team, year = 2020)
```


## Arguments

Argument      |Description
------------- |----------------
`team`     |     Team filter to select.
`year`     |     Year of data to pull


## Examples

```r
kp_minutes_matrix(team = 'Florida St.', year = 2020)
```
