# `espn_mbb_rankings`

## Description

Get men's college basketball AP and Coaches Poll rankings from ESPN


## Usage

```r
espn_mbb_rankings()
```

## Value

Returns a tibble

## Examples

```r
# Get current AP and Coaches Poll rankings
espn_mbb_rankings()
```
