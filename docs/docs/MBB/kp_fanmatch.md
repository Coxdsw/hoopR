# `kp_fanmatch`

## Description

Get FanMatch by date


## Usage

```r
kp_fanmatch(date = "2020-02-12")
```


## Arguments

Argument      |Description
------------- |----------------
`date`     |     Date of games to pull (YYYY-MM-DD)


## Value

A data frame 16 columns

## Examples

```r
kp_fanmatch(date="2020-03-10")
```
