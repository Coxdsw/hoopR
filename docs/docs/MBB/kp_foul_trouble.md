# `kp_foul_trouble`

## Description

Get 2-Foul Participation Stats


## Usage

```r
kp_foul_trouble(min_year, max_year)
```


## Arguments

Argument      |Description
------------- |----------------
`min_year`     |     First year of data to pull
`max_year`     |     Last year of data to pull


## Examples

```r
kp_foul_trouble(min_year = 2020, max_year = 2020)
```
