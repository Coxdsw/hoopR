# `kp_winprob`

## Description

Get Win Probability


## Usage

```r
kp_winprob(game_id, year)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game id of game to pull
`year`     |     Year of game to pull


## Examples

```r
kp_winprob(game_id = 1238, year = 2020)
```
